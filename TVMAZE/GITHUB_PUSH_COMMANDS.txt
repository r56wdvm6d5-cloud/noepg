# GitHub Push Commands for TVMAZE EPG

## Quick Start

### Make Script Executable
```bash
chmod +x github_push.sh
```

### Run the Push Script
```bash
./github_push.sh
```

## Manual Git Commands

### One-time Setup (if needed)
```bash
# Configure git user (replace with your info)
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

# Clone repository manually
git clone https://github.com/r56wdvm6d5-cloud/noepg
cd noepg
```

### Manual Push Steps
```bash
# Navigate to TVMAZE directory
cd /Users/ah/CascadeProjects/windsurf-project/TVMAZE

# Copy EPG file to repository
cp epg.xml noepg_repo/

# Navigate to repository
cd noepg_repo

# Check status
git status

# Add file
git add epg.xml

# Commit changes
git commit -m "Update EPG data - $(date +'%Y-%m-%d %H:%M:%S')"

# Push to GitHub
git push origin main
```

## Force Push (if needed)

### Force Push Latest Commit
```bash
cd noepg_repo
git push --force-with-lease origin main
```

### Hard Reset and Force Push
```bash
cd noepg_repo
git reset --hard HEAD~1  # Remove last commit
git add epg.xml
git commit -m "Force update EPG data"
git push --force origin main
```

## Alternative Methods

### Using GitHub CLI (if installed)
```bash
# Install GitHub CLI first
gh auth login

# Push using gh
cd noepg_repo
gh repo clone r56wdvm6d5-cloud/noepg .
cp ../epg.xml .
git add epg.xml
git commit -m "Update EPG data"
git push
gh repo view
```

### Using curl API (advanced)
```bash
# Requires GitHub personal access token
TOKEN="your_github_token"
REPO="r56wdvm6d5-cloud/noepg"
FILE="epg.xml"

# Get file SHA
SHA=$(curl -H "Authorization: token $TOKEN" \
       -H "Accept: application/vnd.github.v3+json" \
       https://api.github.com/repos/$REPO/contents/$FILE | \
       jq -r .sha)

# Update file
curl -X PUT \
  -H "Authorization: token $TOKEN" \
  -H "Accept: application/vnd.github.v3+json" \
  https://api.github.com/repos/$REPO/contents/$FILE \
  -d "{
    \"message\": \"Update EPG data - $(date +'%Y-%m-%d %H:%M:%S')\",
    \"sha\": \"$SHA\",
    \"content\": \"$(base64 -w 0 epg.xml)\"
  }"
```

## Troubleshooting

### Authentication Issues
```bash
# Check git config
git config --list

# Set up personal access token
git remote set-url origin https://your_token@github.com/r56wdvm6d5-cloud/noepg.git
```

### Repository Access
```bash
# Test repository access
curl -I https://github.com/r56wdvm6d5-cloud/noepg

# Check if repository exists and is accessible
gh repo view r56wdvm6d5-cloud/noepg
```

### Network Issues
```bash
# Test connectivity
ping github.com

# Check git remote
git remote -v
```

## Automation

### Add to crontab for automatic updates
```bash
# Edit crontab
crontab -e

# Add line for every 6 hours
0 */6 * * * cd /Users/ah/CascadeProjects/windsurf-project/TVMAZE && python3 tvmaze_epg.py && ./github_push.sh
```

### Create batch script
```bash
#!/bin/bash
# auto_update_and_push.sh
echo "Starting EPG update and push..."
python3 tvmaze_epg.py
if [ $? -eq 0 ]; then
    echo "EPG generated successfully, pushing to GitHub..."
    ./github_push.sh
else
    echo "EPG generation failed, skipping push."
fi
```
